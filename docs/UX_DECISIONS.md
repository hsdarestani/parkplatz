# UX Decisions

- Benchmarks: EasyPark speed, Ampido private-space confidence, ParkBee guaranteed-space clarity, Uber map/sheet hierarchy, Airbnb listing trust.
- Visual direction: European urban mobility meets precision navigation instrument; porcelain canvas, deep ink, electric mint, amber destination signals.
- Primary flow: launch → discovery map → search sheet → destination/time/vehicle → animated resolution → `/search` results → marker/card preview.
- Wow moment: concise status sequence confirms availability without a fake long wait.
- Responsive rules: mobile uses full map and bottom sheet; desktop uses a 430px side panel plus full-height map; tablet transitions at tokenized breakpoints.
- Product rules: never expose exact address pre-booking, show all-in price, show dimensions/access/verification, keep manual search independent from location permission, no background location.
- Next iteration: full detail route, checkout-safe booking intent, host profile trust, reviews, cancellation policy depth, real map tile integration hardening.
